# Omniauth::Salesloft

An OmniAuth Strategy for Connecting to the SalesLoft Public API.

## Installation

Add this line to your application's Gemfile:

    gem 'omniauth-salesloft'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install omniauth-salesloft
